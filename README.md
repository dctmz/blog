# 团队博客

日拱一卒，保持学习

博客采用[hexo](https://hexo.io/zh-cn/)

## 主题

主题采用 next

### 安装主题

```bash
git clone https://github.com/theme-next/hexo-theme-next themes/next
```

### 配置主题

找到项目下的 `_config.next.yml`文件，编辑即可

## 开发

如果需要进行二次开发，请按照一下步骤

### 安装依赖

```bash
npm i
```

### 启动项目

```bash
npm run dev
```

## 部署

项目采用 GitHub 进行部署，直接运行命令即可部署，命令如下

```bash
npm run cd
```
